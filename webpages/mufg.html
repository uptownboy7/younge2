개요

일본 전체 2위의 재벌그룹 , 미쓰비시 그룹의 카드사 웹사이트 리뉴얼 안건입니다. 에이전시 일본 본사에서 디자인을 제작하여, 한국 지부에서 HTML마크업 코딩을 진행했습니다.

약 750페이지의 HTML파일을 제작했습니다.
해당 안건은 일본 본사로부터 디자인을 스스로 수정할 수 있는 디자이너 겸 퍼블리셔가 참가하길 바라고 있었기 때문에 한국 지부에서는 제가 혼자 대응하게 되었습니다.

━Summary　

일본의 3대 재벌 기업중 하나인 미쓰비시 그룹
(Mitsubishi UFJ Financial Group, Inc.)의 카드사 웹사이트 리뉴얼 안건입니다.
카드 정보, 카드 신청 등 800페이지 이상 대량의 웹페이지 새로운 디자인으로 코딩 하는 역할을 맡았습니다.

해당 리뉴얼은 크게 세 가지 과정으로 진행 됐습니다.

Work Procedure

　

　

━Analysis　

현행 사이트의 코드를 열어 분석합니다.
컴포넌트가 어떻게 만들어져있는지, 또 어떻게 하면 수정을 최대한 줄여서 새버전으로 리뉴얼 할지 분석합니다.

특히, 리뉴얼 웹사이트와 클래스명이나 아이디가 겹치면 안되기 때문에 주의를 기울여서 조사합니다.

　

코드 분석

상기 페이지에는 로그인, 아코디언 파츠 등이 포함되어있습니다.

로그인 파츠: ssi로 삽입 된 방식
아코디언: html,css로 구현, js로 animate효과

직접적으로 코드를 볼 수 있는 아코디언을 리뉴얼 해보겠습니다.

━Make Component　

분석 내용과 XD 디자인을 토대로, 리뉴얼 컴포넌트를 제작합니다.

XD 디자인 분석

에이전시 재직 당시, 디자인, 코딩 양쪽 역량을 인정받아
디자이너 겸 퍼블리셔로 겸직을 맡았습니다.
때문에 따로 제플린으로 전환하지 않아도 XD 자체로 디자인 분석이
가능하며, XD 디자인을 즉시 웹개발에 응용할 수 있습니다.

컴포넌트 제작

XD디자인과 구버전 코드를 토대로 리뉴얼 컴포넌트 를 제작합니다. 아코디언이 닫혀있을 때 「+」모양, 열려 있을 때 「-」 모양으로 바뀌는 인터렉션을 javascript로 구현합니다.

기본 문법에 충실한 코딩

추후 다른 개발자가 마크업과 스크립트를 수정할 경우가 생겨도
코드를 쉽게 알아보고 수정할 수 있도록 필요한 내용만 간단하게 작성했습니다.

━Mass-Product

완성된 컴포넌트를 토대로, 페이지를 대량 양산합니다.

컴포넌트 완성

아코디언뿐만 아니라, mufg웹사이트 내에 존재하는 모든 디자인을 컴포넌트화하여 페이지 대량 제작을 준비합니다.

페이지 대량 양산

모든 컴포넌트의 제작이 끝나면, 모든 페이지의 디자인을 리뉴얼 버전의 디자인으로 만드는 작업을 진행합니다.

━Mass-Product

페이지 레이아웃 제작이 끝나면, javascript 동적 효과를 검사합니다.

━Perfection

약 750페이지의 html, css, js 스크립트가 완성되었으며,
Quality Check 이후 고객(mufg)에게 전달됩니다.
